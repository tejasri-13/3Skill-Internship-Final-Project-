# Event Management System

## 📌 Project Overview

The **Event Management System** is a full-stack web application developed as the final project for the **3SKILL Internship**. The system simplifies event organization by allowing organizers to create and manage events while enabling users to book tickets securely. It also includes QR code-based check-in, payment integration, and an analytics dashboard for monitoring event performance.

---

## 🚀 Features

- 🎉 Event Creation and Management
- 🎟️ Online Ticket Booking
- 📱 QR Code-Based Event Check-in
- 💳 Secure Payment Integration
- 📊 Analytics Dashboard
- 👤 User Authentication
- 📅 Event Scheduling
- 📧 Booking Confirmation

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/event-management-system.git
```

### Navigate to the Project

```bash
cd event-management-system
```

### Install Dependencies

For Backend:

```bash
cd server
npm install
```

For Frontend:

```bash
cd client
npm install
```

---

## ▶️ Run the Project

### Start Backend

```bash
npm start
```

### Start Frontend

```bash
npm start
```

The application will run on:

- Frontend: `http://localhost:3000`
- Backend: `http://localhost:5000`
